Korzystałem z Python 3.10

Wykorzystałem zewnętrzną bibliotekę do grafów networkx - jej zainstalowanie jest konieczne do działania programu.
Z tej biblioteki wykorzystałem strukturę grafu, metodę zwracającą wierzchołki bez wchodzących krawędzi i metode
wykrywającą cykle - wszystko to już implementowaliśmy, więc działałem w myśl zasady nie wymyślania koła na nowo.

Program przyjmuje jako argument wywołania plik testowy, przykładowe wywołanie (windows)
->python main.py test-basic.txt
